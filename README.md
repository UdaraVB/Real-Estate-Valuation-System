# 🏠 Karachi House Price Predictor (MLOps)

![Machine Learning CI](https://github.com/UdaraVB/Real-Estate-Valuation-System/actions/workflows/ml_ci.yml/badge.svg)

An end-to-end Machine Learning pipeline for predicting real estate prices in Karachi, Pakistan. This project demonstrates professional MLOps practices, from data cleaning to automated testing.

## 🚀 Key Features
* **Custom Preprocessing:** Handles Karachi-specific price formats (Crore/Lakh) and Square Yard conversions.
* **Random Forest Model:** Achieved a strong $R^2$ score of **0.83** on test data.
* **MLOps Pipeline:** Experiment tracking via **MLflow** and automated testing with **GitHub Actions**.
* **Clean Code:** Adheres to PEP8 standards with automated linting (Flake8).
* **Interactive UI:** A user-friendly **Streamlit** dashboard for instant property valuation.

## 📊 Performance
* **Model:** Random Forest Regressor
* **Accuracy:** $R^2 = 0.83$
* **Validation:** Automated unit tests for data cleaning logic (Pytest).

## 🛠️ Tech Stack
* **Language:** Python 3.11
* **ML Libraries:** Scikit-Learn, Pandas, NumPy
* **Tracking:** MLflow
* **Frontend:** Streamlit
* **CI/CD:** GitHub Actions
* **Environment:** Virtual Environments / Requirements.txt
